# tunning

`tunning` is a small logging helper for CLI applications. It builds on Python's
stdlib `logging` package and Rich to provide styled console output, YAML-driven
configuration, rotating file logs, custom levels, and a styled prompt helper.

Use these pages for package-level documentation:

- [Quick Start](quick-start.md): basic setup and first log messages.
- [Configuration](configuration.md): programmatic and YAML configuration notes.
- [Examples](examples.md): practical snippets.
- [API Reference](api.md): generated from public source docstrings.

The README remains the short project overview for GitHub and PyPI. The API
reference is generated from source docstrings so it stays close to the public
runtime surface.
